# LeadCapture Page — Modern Email Waitlist Landing Page

A clean and modern landing page built using HTML and CSS, designed to collect email signups for a product launch or startup waitlist.

This project focuses on creating a visually appealing dark-themed UI with a minimal layout, smooth hover effects, and a conversion-focused design approach.

---

## Features

- Dark modern aesthetic
- Email signup section
- Animations on Content
- Smooth button hover animations
- Centered hero content
- Clean typography and spacing
- Beginner-friendly project structure

---

## Tech Stack

- HTML5
- CSS3

---

## Project Goal

The purpose of this project was to practice building modern landing pages while improving layout design, styling, and UI presentation.
